ERPBridge

O ERPBridge é uma plataforma de integração entre sistemas ERP. O aplicativo permite que usuários realizem autenticação segura utilizando Google OAuth para acessar funcionalidades internas da aplicação.

Finalidade do Aplicativo

Este aplicativo utiliza autenticação do Google exclusivamente para permitir login seguro dos usuários. Nenhum dado é vendido, compartilhado ou utilizado para fins comerciais.
